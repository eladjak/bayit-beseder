בעיות שנמצאו

▼
  • ChunkLoadError בדשבורד: Cache ישן אחרי דפלויי — נפתר עם hard reload. לא באג אמיתי.
  • node_modules נמחקו: ניקוי דיסק אוטומטי (10GB free) מחק את התלויות. צריך npm install לפני build.
  • TypeScript גלובלי קונפליקט: npx tsc לקח גרסה 6.0.2 במקום local — נפתר עם npm run build ישיר.
