## 2. פסיכולוגיית תמחור בישראל

### ממצאים מרכזיים

1. **ישראלים רגישים למחיר** — סבירות של 2.5 פעמים יותר מהממוצע העולמי להיות "חסכנים", ו-50–60% פחות נכונות לשלם מחירי פרימיום ([GWI](https://blog.gwi.com/chart-of-the-week/profiling-consumers-in-israel/))

2. **מספרים עגולים עדיפים** — רשויות רגולטוריות בישראל אסרו מחירים שמסתיימים ב-99 (כמו ₪19.99). בנוסף, הטבעות קטנות יצאו ממחזור, מה שהופך מחירים כמו ₪19.90 ללא רלוונטיים. **עדיף מספרים עגולים: ₪15, ₪20, ₪10.**

3. **"חודשיים חינם" > "הנחה של 30%"** — צרכנים ישראלים מגיבים טוב יותר למסגור קונקרטי ("2 חודשים מתנה") מאשר לאחוזים

4. **שנתי מועדף עם הנחה משמעותית** — 59% מהמנויים מעדיפים מסלול שנתי כשההנחה היא 30–40%

5. **LTV גבוה** — ישראל מדורגת מקום 3 עולמית ב-LTV מנויי אפליקציות ($27), אחרי שוויץ וקטאר. משמעות: ישראלים *שמחליטים לשלם* — משלמים ונשארים.

## 3. שיעורי המרה (Freemium Conversion)

### סטטיסטיקות ענפיות

| מדד | ערך |
|------|-----|
| המרה ממוצעת freemium → paid (כלל SaaS) | 3–5% |
| אפליקציות פרודוקטיביות | 3–7% |
| אפליקציות צרכניות (כלליות) | 1–2% |
| Top quartile | 8–15% |
| Hard paywall (אין חינמי) | 15–25% (אבל פחות הורדות) |

### מה זה אומר לבית בסדר?

עם **קהל יעד צר** (זוגות ישראלים, לא שוק מאסי) ו**ערך ברור** (ניהול בית משותף), ניתן לצפות ל:
- **ציפייה ריאלית:** 3–5% המרה
- **עם אופטימיזציה (trial, onboarding חכם):** 5–8%
- **יעד שאפתני:** 10%+

### מפתח להמרה גבוהה:
- ❌ אל תתנו *הכל* בחינם — אחרת אין סיבה לשדרג
- ✅ תנו מספיק כדי להתמכר, אבל עם חיכוך מורגש
- ✅ הציגו את הפרימיום ברגע הנכון (Contextual upgrade prompts)
- ✅ ברירת מחדל על "שנתי" בעמוד התמחור

## 4. אסטרטגיית Feature Gating — מה חינמי ומה בתשלום

### עקרון מנחה: "כלל 60/40"

> 60% מהערך בחינם (מספיק להתמכר), 40% בפרימיום (מספיק להצדיק תשלום)

### טבלת חלוקת פיצ'רים

| פיצ'ר | חינם 🆓 | פרימיום ⭐ | הסבר |
|--------|---------|-----------|-------|
| **משימות בסיסיות** | ✅ עד 20 משימות | ✅ ללא הגבלה | 20 מספיק לטעום, לא מספיק לנהל בית שלם |
| **2 משתמשים** | ✅ | ✅ | זה הקונספט — תמיד 2 |
| **קטגוריות ברירת מחדל** | ✅ 8 קטגוריות | ✅ קטגוריות מותאמות אישית | מותאם אישית = ערך פרימיום |
| **רשימת קניות** | ✅ רשימה אחת | ✅ רשימות מרובות + היסטוריה | רשימה אחת בסיסית, מרובות לפרימיום |
| **תוכנית שבועית** | ✅ ידנית בלבד | ✅ חלוקה אוטומטית (wizard) | ה-wizard האוטומטי הוא ה-hook העיקרי |
| **גרירה ושחרור (D&D)** | ✅ | ✅ | פיצ'ר UX בסיסי, לא לחסום |
| **יחס הזהב** | ✅ תצוגה בסיסית | ✅ טרנדים + ניתוח מעמיק | הנתון הבסיסי חינם, ניתוח = פרימיום |
| **סטטיסטיקות** | ❌ | ✅ מלא | זה פיצ'ר "רוצה עוד" מובהק |
| **גיימיפיקציה** | ✅ סטריק בסיסי | ✅ הישגים + badges + מסלולים | סטריק ממכר חינם, שאר הגיימיפיקציה בפרימיום |
| **אימון (coaching)** | ❌ | ✅ טיפים יומיים + מוטיבציה | תוכן בלעדי לפרימיום |
| **מצב עונתי (פסח וכו')** | ❌ | ✅ תבניות עונתיות | תוכן מובנה עם ערך גבוה |
| **יצוא נתונים** | ❌ | ✅ | פיצ'ר power-user |
| **ערכות נושא** | ❌ | ✅ ערכות צבע + מצב כהה מותאם | Cosmetic premium |
| **התראות WhatsApp** | ❌ | ✅ תזכורות + עדכונים | עלות תפעולית, מצדיק פרימיום |
| **זונות (Zone scheduling)** | ❌ | ✅ | פיצ'ר מתקדם עתידי |
| **מודעות** | כן (באנר קטן) | ❌ ללא מודעות | הכנסה מחינמיים + מוטיבציה לשדרוג |

### Contextual Upgrade Prompts (חשוב!)

בכל פעם שמשתמש חינמי נוגע בפיצ'ר נעול:
```
"רוצה לראות סטטיסטיקות מפורטות? 📊
שדרגו לבית בסדר+ וגלו מי באמת עושה יותר בבית!
נסו חינם ל-7 ימים →"
```

## 5. המלצת תמחור

### מבנה מוצע

```
┌─────────────────────────────────────────────────────┐
│                    בית בסדר                          │
│                   (חינם לנצח)                        │
│                                                      │
│  ✅ 20 משימות  ✅ 2 משתמשים  ✅ רשימת קניות          │
│  ✅ D&D  ✅ סטריק בסיסי  ✅ תוכנית שבועית ידנית      │
│  ⚠️ מודעות קטנות                                    │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│                  ⭐ בית בסדר+                        │
│              ₪15/חודש  |  ₪120/שנה                   │
│              (= ₪10/חודש, חיסכון של 33%)             │
│                                                      │
│  🔓 משימות ללא הגבלה  🔓 סטטיסטיקות מלאות           │
│  🔓 חלוקה אוטומטית  🔓 אימון + מוטיבציה             │
│  🔓 מצבים עונתיים  🔓 קטגוריות מותאמות               │
│  🔓 הישגים + badges  🔓 ללא מודעות                   │
│  🔓 התראות WhatsApp  🔓 זונות (בקרוב)                │
└─────────────────────────────────────────────────────┘
```

### הנמקה

| החלטה | סיבה |
|-------|------|
| **₪15/חודש** (לא ₪19.90) | ישראלים מעדיפים מספרים עגולים. ₪15 נתפס כ"פחות מכוס קפה ביום". ₪20 כבר גבוה מדי לאפליקציית ניהול בית. ₪15 תואם את Life Plus ו-MyFinanda. |
| **₪120/שנה** (לא ₪149) | ₪120 = ₪10/חודש = הנחה של 33%. מספר עגול ופסיכולוגית "מתחת ל-₪150". תואם את טווח ה-30–40% הנחה שמניע 59% מהמנויים לבחור שנתי. |
| **"חודשיים חינם"** | עדיף לשווק כ"קבלו 4 חודשים מתנה!" (₪180 אלטרנטיבה לעומת ₪120) מאשר "33% הנחה". |
| **20 משימות (לא 15)** | 15 משימות זה צפוף מדי — משתמשים ירגישו מוגבלים לפני שהתמכרו. 20 מספיק לחוויה טובה, אבל בית ממוצע צריך 30–50. |
| **Trial 7 ימים** | ממוצע הענף ל-consumer apps. מספיק לחוות שבוע שלם (כולל שבת). |

## 6. אסטרטגיית השקה

### שלב 1: Founder's Pricing (חודשים 1–3)

```
"מייסדים — מחיר מיוחד לנצח!"

₪9.90/חודש  |  ₪79/שנה (= ₪6.58/חודש)
ל-100 המנויים הראשונים בלבד
```

**למה:**
- יוצר דחיפות (FOMO)
- בונה בסיס משתמשים משלם
- מפתה early adopters שיעזרו עם פידבק
- "לנצח" = נאמנות + שימור

### שלב 2: Early Bird (חודשים 3–6)

```
"הנחת השקה — עוד מעט נגמר!"

₪12/חודש  |  ₪99/שנה (= ₪8.25/חודש)
```

### שלב 3: מחיר מלא (מחודש 6+)

```
₪15/חודש  |  ₪120/שנה (= ₪10/חודש)
```

### טיפ: Price Anchoring

בעמוד התמחור, תמיד להציג את המחיר השנתי **ליד** החודשי:

```
        חודשי           שנתי (הכי משתלם!)
       ₪15/חודש    →    ₪10/חודש (₪120/שנה)
                        חוסכים ₪60 בשנה! 🎉
```

ברירת המחדל = שנתי (toggle מסומן על שנתי).

## 7. טיפים לאופטימיזציית המרה

### 7.1 Upgrade Triggers — מתי להציע שדרוג

| טריגר | הודעה |
|-------|-------|
| משתמש הגיע ל-18/20 משימות | "כמעט מלא! שדרגו ל-+ כדי להוסיף משימות ללא הגבלה" |
| לחיצה על "סטטיסטיקות" (נעול) | "רוצה לדעת מי עושה יותר? נסו + חינם ל-7 ימים" |
| סוף שבוע ראשון | "השבוע הראשון שלכם! שדרגו ל-+ לחלוקה אוטומטית" |
| לפני פסח/חג | "מצב פסח זמין רק ב-+! 37 משימות מוכנות לניקיון" |
| סטריק של 7 ימים | "7 ימים ברצף! שדרגו ל-+ לפתוח הישגים ובאדג'ים" |
| 30 ימים שימוש חינם | "אתם משתמשים נאמנים! הנה הנחה בלעדית: ₪9.90/חודש" |

### 7.2 שימור (Retention)

- **תזכורת לפני חידוש:** "המנוי שלכם מתחדש בעוד 3 ימים"
- **Win-back:** אם ביטלו, אחרי 30 יום: "חסר לנו! חזרו ב-₪9.90/חודש לחודשיים"
- **Annual lock-in:** תמיד לדחוף שנתי — churn נמוך פי 3 מחודשי

### 7.3 שיטות תשלום

| שיטה | עדיפות | הערה |
|------|--------|------|
| Apple In-App Purchase | ⭐⭐⭐ | 30% עמלה, אבל אמון גבוה + קל |
| Google Play Billing | ⭐⭐⭐ | 15% עמלה (Small Business Program) |
| Stripe (חיוב ישיר) | ⭐⭐ | 2.9% + ₪1, ללא עמלת חנות |
| PayPlus / Tranzilla | ⭐ | ישראלי, אבל פחות מוכר ב-apps |

**המלצה:** להתחיל עם **App Store + Google Play** (אמון, קלות) ובמקביל Stripe דרך האתר (מחיר מוזל "אקסקלוסיבי לאתר" ב-₪12/חודש כדי לעקוף עמלות).

## 8. סיכום — לוח המלצות

| פרמטר | המלצה |
|-------|-------|
| **מחיר חודשי** | ₪15 |
| **מחיר שנתי** | ₪120 (= ₪10/חודש) |
| **הנחה שנתית** | 33% ("4 חודשים חינם!") |
| **Founder's Pricing** | ₪9.90/חודש, ₪79/שנה (100 ראשונים) |
| **Trial** | 7 ימים חינם |
| **מגבלת חינם** | 20 משימות, רשימה אחת, ללא סטטיסטיקות |
| **Feature gate עיקרי** | סטטיסטיקות + Wizard אוטומטי + אימון |
| **ברירת מחדל** | שנתי (toggle) |
| **שפה שיווקית** | "4 חודשים מתנה" > "33% הנחה" |
| **המרה צפויה** | 3–5% (יעד: 8%) |
| **ROI breakeven** | ~200 מנויים שנתיים = ₪24,000/שנה |
